# generatingDocExample

## Route
| Route | Method | Params | Description |
| --- | --- | --- | --- |
| gen/docpdf | POST | text | html string to convert to pdf |

## Silex
```
cd server
composer install
composer run
```

By default on `http://localhost:8888/`



## Angular2
```
cd client/angular2
npm install
ng serve
```

By default on `http://localhost:4200/`
